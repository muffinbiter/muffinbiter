# 🧪 Design Focus

My experiments explore three main questions:

**1. AI Characters**  
How can AI-driven personalities create more dynamic NPC interaction?

**2. AI-Generated Gameplay**  
How can AI produce unique situations, stories, or mechanics every playthrough?

**3. Player Creativity + AI**  
How can AI turn player input (drawing, dialogue, decisions) into gameplay systems?
